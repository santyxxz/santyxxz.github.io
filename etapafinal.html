Etapa post-extracción

Evaporación

Consiste en la eliminacion de la mayor parte de contenido inicial de el agua y se puede utilizar como base para hacer bebidas, se puede mezclar con otros jugos o puede ser consumido directamente mezclado con jugo.

Almacenamiento

Luego del la evaporacion el zumo queda concentrado y es almacenado, este queda listo para realizar una "mezcla".

Mezcla

Se agrega principalmente compuestos como: saborizantes, conservantes, especies, agentes espesador, agua sal, jarabes o azúcares. Para la mezcla tambien se le agrega un "plus" jugo obtenido en el proceso de (prensado) para matener aroma y sabor.

Pausterización

Es un método de conservación de jugos y concentrados que consiste en calentar el jugo a una temperatura de 60-100°c durabte un tiempo variable.
Tiene como objetivo:
-Inactivar enzimas
-Eliminar microorganismos patogenos.

Inspeccion final y envasado

Luego de realizar una inspeccion final el jugo es envasado y queda listo para transportar.
